# Espacio Contextual - Web (Astro)

Este es el sitio web oficial de **Espacio Contextual**, reconstruido desde cero utilizando **Astro (SSG)** para una máxima velocidad, SEO y flexibilidad.

## Requisitos
- Node.js (v18+)
- npm

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## CMS
El sitio utiliza **Decap CMS**. Puedes acceder al panel de administración en `/admin`.
La configuración se encuentra en `public/admin/config.yml`.

## Despliegue
Preparado para desplegar en **Netlify**. La configuración se encuentra en `netlify.toml`.
- Comando de construcción: `npm run build`
- Carpeta de publicación: `dist`
